# HolyFetch
# Fetch to more!
![1 Screen](https://github.com/wholos/HolyFetch/blob/main/holyfetch.png)

How to Install!?
``` bash
sudo pacman -S git clang
git clone https://github.com/wholos/HolyFetch
cd HolyFetch/
clang main.c -o main.out
sudo cp main.out /usr/local/bin/holyfetch
```
Ready!

How to use!?
``` bash
holyfetch
```
